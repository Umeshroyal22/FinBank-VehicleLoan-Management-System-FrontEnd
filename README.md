🚗 FinBank Vehicle Loan Management System – Frontend
📌 Overview

The FinBank Vehicle Loan Management System Frontend is a modern Angular-based web application that provides an intuitive interface for customers to apply for vehicle loans and track their applications.

The application communicates with backend microservices through secure REST APIs and provides a seamless digital loan application experience.

🎯 Frontend Features
🏠 Dashboard

Displays overview of loan services

Quick navigation to apply for loans

Shows important loan information and announcements

🚗 Loan Type Selection

Users can select different vehicle loan types

Displays available loan options

Easy navigation to loan application form

📝 Loan Application Form

Customers can apply for vehicle loans online

Captures important details such as:

Personal Information

PAN details

Loan Amount

Tenure

Form validation for accurate data entry

👀 Review & Submit

Users can review all entered loan details

Confirm and submit loan application

📊 Application Status Tracking

Customers can track their loan application

Displays current loan status such as:

Pending

Approved

Rejected

📂 Loan History

Displays list of previous loan applications

Shows loan details and application history

🔎 Loan Details Page

Detailed view of selected loan

Shows:

Loan Amount

Interest Rate

Tenure

Loan Status

🔐 Secure Login System

Authentication using JWT token

Secure access to application features

Mobile-friendly design

Works on desktop, tablet, and mobile devices
